Title: Package File

----

Text:

A file that contains software, such as an [application](/terms/application) program or [extension](/terms/extension), which can be transferred to a Newton device using a [package installer](/terms/package-installer). Denoted by the [.pkg](/terms/pkg) or [.newtonpkg](/terms/newtonpkg) file extension.
