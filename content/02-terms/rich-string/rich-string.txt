Title: Rich String

----

Text:

A string [object](/terms/object) that contains embedded [ink words](/terms/ink-word). Rich strings create a compact representation for strings that contain ink words and can be used with most of the string processing [functions](/terms/function) provided in the [Newton OS](/terms/newton-os).
