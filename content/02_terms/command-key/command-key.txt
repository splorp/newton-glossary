Title: Command Key

----

Date: 19 May 2022

----

Text:

A command key associates a programmatic (link: /terms/message text: message) or (link: /terms/action text: action) with a (link: /terms/key-combination text: key combination). The key combination consists of a (link: /terms/character text: character) typed on a keyboard in combination with one or more (link: /terms/modifier-key text: modifier keys).

The Command key is also the name of a modifier key found on a keyboard.

On an (link: /terms/emate-300 text: eMate 300) keyboard, the Command keys on either side of the space bar are identified by an (link: /terms/icon text: icon) that looks like a (link: /terms/propeller text: propeller) or (link: /terms/cloverleaf text: cloverleaf) &#x2318; (officially known by its (link: /terms/unicode text: Unicode) name as a _“place of interest sign”_) and an (link: /terms/open-apple text: Open Apple) icon that looks like an outlined version of the [Apple](https://www.apple.com/) logo.

The Command key on a (link: /terms/newton-keyboard text: Newton Keyboard) has only an Open Apple icon.
