Title: Password

----

Date: 10 February 2024

----

Text:

A unique word consisting of several (link: /terms/character text: characters) that allows secure access to a system or device. On a Newton device, a password is entered using a (link: /terms/password-slip text: password slip).

(image: /assets/images/password-slip-2x.gif alt: Screenshot of the Password Slip width: 241)

Password is abbreviated as (link: /terms/pswd text: pswd) in the password slip when confirming the creation of a new password or updating an existing password in the (link: /terms/change-password-slip text: change password slip).

A password is different from a (link: /terms/pin text: PIN) which consists entirely of numbers.
