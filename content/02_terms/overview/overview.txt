Title: Overview

----

Date: 27 August 2018

----

Text:

A (link: /terms/roll-view text: roll view) that displays a summarized list of multiple items. From an overview, a user can either (link: /terms/tap text: tap) an item to see more detailed information or select, view, and interact with multiple items by (link: /terms/tapping text: tapping) a (link: /terms/checkbox text: checkbox) beside each item and performing an action, such as (link: /terms/routing text: routing).

An overview is displayed by tapping the (link: /terms/overview-button text: overview button) in an (link: /terms/application text: application) such as (link: /terms/notepad text: Notepad) or the (link: /terms/extras-drawer text: Extras Drawer). On an (link: /terms/emate-300 text: eMate 300), an overview can be displayed by pressing the (link: /terms/overview-key text: Overview key) on the keyboard.
