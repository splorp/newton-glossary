Title: Pointer Tool

----

Date: 17 February 2026

----

Text:

A tool found in (link: /terms/newton-works text: Newton Works) used to select, move, and resize objects in a (link: /terms/drawing text: drawing) document.

When a new drawing document is created, the pointer tool is (link: /terms/highlight text: highlighted) by default.
