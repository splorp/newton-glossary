Title: Change Password Slip

----

Date: 16 April 2026

----

Text:

A (link: /terms/slip text: slip) that allows the user to update an existing (link: /terms/password text: password) without echoing the password in plain text. Typed (link: /terms/character text: characters) appear as bullets in the (link: /terms/input-line text: input lines).

(image: /assets/images/change-password-slip-2x.gif alt: Screenshot of the Change Password slip with a typewriter keyboard width: 241)

When displayed on an (link: /terms/emate-300 text: eMate 300) or when a physical keyboard such as the (link: /terms/newton-keyboard text: Newton Keyboard) is connected to a Newton device, the change password slip does not include a (link: /terms/typewriter text: typewriter) keyboard.

Password is sometimes abbreviated as (link: /terms/pswd text: pswd) when displayed in a slip.
