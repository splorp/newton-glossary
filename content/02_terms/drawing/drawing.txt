Title: Drawing

----

Date: 01 January 2018

----

Text:

The document format (or (link: /terms/stationery text: stationery)) used by the (link: /terms/draw text: Draw) component of the (link: /terms/newton-works text: Newton Works) application.

(image: /assets/images/drawing-2x.gif alt: Screenshot of a Drawing document in Newton Works width: 482)

When a new drawing document is created, it is assigned the default title “(link: /terms/untitled-drawing text: Untitled Drawing)”, which can be changed in the (link: /terms/title slip text: title slip).

Drawing stationery is called (link: /terms/draw-paper text: Draw Paper) in the title slip.
