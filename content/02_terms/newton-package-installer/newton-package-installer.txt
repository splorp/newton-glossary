Title: Newton Package Installer

----

Date:

----

Text:

Software developed by [Apple](https://www.apple.com/) that allows the user to install [packages](/terms/package) on a Newton device from a [desktop computer](/terms/desktop-computer) or [laptop](/terms/laptop-computer).

Also referred to as [NPI](/terms/npi).
