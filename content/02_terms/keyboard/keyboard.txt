Title: Keyboard

----

Date: 28 December 2017

----

Text:

A set of keys used to input (link: /terms/character text: characters) into a computer.

(link: /terms/newton-os text: Newton OS) supports multiple onscreen keyboards for entering different types of data. Selecting a keyboard is accomplished by (link: /terms/tapping text: tapping) the (link: /terms/keyboard-button text: keyboard button).

The (link: /terms/emate-300 text: eMate 300) also has a physical keyboard built into its case.
