Title: Folder Slip

----

Date: 21 January 2018

----

Text:

A [slip](/terms/slip) listing currently available storage locations, such as the [internal store](/terms/internal-store) and a named [external store](/terms/external-store), where selected items can be filed. The folder slip also lists the [folders](/terms/folder) defined for the current [application](/terms/application) and provides [buttons](/terms/button) for creating new folders and editing existing folder names.

Also referred to as a [Filing Slip](/terms/filing-slip).
