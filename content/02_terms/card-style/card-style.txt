Title: Card Style

----

Date: 14 June 2026

----

Text:

The appearance or layout of a (link: /terms/card text: card) view in the (link: /terms/name-file text: Name File) application.

Various style options are available in the Card Style (link: /terms/slip text: slip), displayed as set of (link: /terms/picture-radio-button text: picture radio buttons), with the selected style indicated by a (link: /terms/thick-border text: thick border).

(image: /assets/images/card-style-2x.gif alt: Screenshot of the Card Style slip width: 320)

The (link: /terms/newton-programmers-guide text: Newton Programmer’s Guide) describes the card styles as:

- Layout with a (link: /terms/squiggly-line text: squiggly line)
- Plain layout
- Layout with dashed lines
- Layout with crossed lines
- Layout with (link: /terms/bullet-holes text: bullet holes) all over
- Layout with (link: /terms/dotted-fading-line text: dotted fading line)
- Layout with big bullet holes in a single line
