Title: Rectangle Tool

----

Date: 16 June 2026

----

Text:

A tool found in (link: /terms/newton-works text: Newton Works) used to add rectangular (link: /terms/object text: objects) to a (link: /terms/drawing text: drawing) document.

(link: /terms/dragging text: Dragging) the rectangle tool on the (link: /terms/screen text: screen) using the (link: /terms/pen text: pen) creates an (link: /terms/rectangle text: rectangle). To create a (link: /terms/square text: square), hold down the (link: /terms/shift-key text: Shift key) while dragging the tool.

(link: /terms/tapping text: Tapping) on the rectangle with the (link: /terms/pointer-tool text: pointer tool) causes a (link: /terms/selection-box text: selection box) to appear around the object, allowing the user to move the object or resize the object using the (link: /terms/handle text: handles).

Also referred to as the (link: /terms/square-tool text: Square Tool).
