Title: Newton User Interface

----

Date: 24 May 2001

----

Text:

The standard rules and conventions for interacting with Newton devices, as set out in the [Newton User Interface Guidelines](/sources/newton-user-interface-guidelines) documentation. The [user interface](/terms/user-interface) provides a consistent means of interacting with all Newton devices and any [applications](/terms/application) designed to run on those devices.
