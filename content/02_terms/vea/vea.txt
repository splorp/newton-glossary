Title: Vea

----

Date: 27 December 2025

----

Text:

Software developed by (link: /sources/planet-computing text: Planet Computing) that featured a set of diagramming tools similar to the (link: /terms/draw text: Draw) component of (link: /terms/newton-works text: Newton Works). In addition to allowing the user to select, position, and layer graphical (link: /terms/object text: objects) using templates for various planning tasks, Vea also provided an underlying (link: /terms/database text: database) which could be used to assign multiple properties to each object.
