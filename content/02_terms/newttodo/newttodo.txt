Title: newtToDo

----

Date: 09 November 2017

Text:

Software developed by (link: /sources/adam-tow text: Adam Tow) that allows the user to efficiently manage (link: /terms/task text: tasks) using a Newton device. Features include reminders with (link: /terms/alarm text: alarms), repeating tasks, additional (link: /terms/priority text: priority) levels, flexible options for sorting and filing, a customizable (link: /terms/user-interface text: user interface), and the ability to automatically import (link: /terms/to-do-list text: To Do List) tasks from the (link: /terms/date-book text: Date Book) application.
