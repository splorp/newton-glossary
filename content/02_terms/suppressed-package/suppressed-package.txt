Title: Suppressed Package

----

Date: 15 April 2026

----

Text:

A (link: /terms/package text: package) that has been deactivated temporarily by the (link: /terms/operating-system text: operating system).

When a Newton device is (link: /terms/reset text: reset) or a (link: /terms/card text: card) is inserted, any suppressed packages will be reactivated. A suppressed package can also be activated by (link: /terms/tapping text: tapping) the package (link: /terms/icon text: icon).

When a package is suppressed, the icon is displayed with an ‘X’.

In comparison, a (link: /terms/frozen-package text: frozen package) is deliberately made inactive by the user using a utility to (link: /terms/freeze text: freeze) the package. A frozen package remains inactive until the user deliberately (link: /terms/thaw text: thaws) the package.

Also referred to as a (link: /terms/deactivated-package text: deactivated package) or (link: /terms/inactive-package text: inactive package).
