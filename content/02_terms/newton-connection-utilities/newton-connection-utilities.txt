Title: Newton Connection Utilities

----

Date:

----

Text:

Software developed by [Apple](https://www.apple.com/) that allows the user to exchange and [synchronize](/terms/synchronize) information between a Newton device and common desktop [applications](/terms/application). This software also offers support for installing [packages](/terms/package) and using a physical keyboard as an input peripheral. The Newton Connection Utilities is for use with [Newton OS](/terms/newton-os) 2.0 and later. The equivalent software package for earlier versions of Newton OS is [Newton Connection](/terms/newton-connection).

The code name for Newton Connection Utilities was [Gyro](/terms/gyro).

Also referred to as [NCU](/terms/ncu).
